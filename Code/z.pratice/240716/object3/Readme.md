상속받은 Object 클래스의 메소드를 오버라이딩하여 프로그램을 작성한다.

3개의 User 객체를 생성하여 배열에 셋팅한 후 각 객체의 모든 정보를 처리 출력한다.

users list -------------------------------------------------------------------

user02 pass02 이영희 25 F 010-5555-7777

user03 pass03 황짂이 20 F 010-9874-5632
